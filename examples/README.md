# Examples

**This folder contains example magic squares of order 3 to 125.** Output is converted to [Fr√©nicle standard form](https://en.wikipedia.org/wiki/Fr%C3%A9nicle_standard_form#:~:text=A%20magic%20square%20is%20in,in%20%5B2%2C1%5D.) to distinguish rotated and transposed duplicates.
